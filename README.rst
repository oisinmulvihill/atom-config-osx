ATOM Settings on OSX
====================


Installing a package
--------------------

Install using from ~/.atom directory. This will also back up the packge
installed for later restoration.

.. code-block:: bash

	# E.g.
	make install PKG=minimap


Package Backup / Restoration
----------------------------

From ~/.atom directory.


Backup
~~~~~~

.. code-block:: bash

	make backup


Restore
~~~~~~~

.. code-block:: bash

	make restore
